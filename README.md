## üíª For Developers (Running Locally)

If you want to contribute or run the code on your own machine:

**Prerequisites:** Node.js

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/CaseyJames669/Sprinkler-Fitter-Journeyman-Exam-Prep-MN-Edition.git](https://github.com/CaseyJames669/Sprinkler-Fitter-Journeyman-Exam-Prep-MN-Edition.git)
    cd Sprinkler-Fitter-Journeyman-Exam-Prep-MN-Edition
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Set up Environment Variables:**
    Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key.
4.  **Run the app:**
    ```bash
    npm run dev
    ```
